# OpenDeepResearcher

## Description
A multi-agent AI research system with ChatGPT-like UI.

## Features
- Planner Agent
- Search Agent (Tavily)
- Writer Agent
- Streamlit UI
- Session Memory
- Multi-chat support

## Milestones
- Milestone 2: Agent pipeline
- Milestone 3: UI + Memory
